# e-comerce front.
> Projeto da matéria Ciência da Computação unifacs 2019.2, criação de um e-comerce.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```

### fixes files
```
yarn run fix
```
